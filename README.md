# EduStep

## Description:
L’application propose des leçons éducatives sous forme de courtes vidéos accompagnées de quiz interactifs pour vérifier la compréhension. Si l’enfant réussit le quiz, il passe au niveau suivant. Sinon, il retourne visionner la vidéo pour mieux comprendre la leçon. EduStep rend l’apprentissage amusant, facile et motivant.
